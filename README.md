## Dependency.
* The required Python packages are listed in requirement.txt

* The FEniCS/Dolfinx package is needed in this project. You can find the installation instructions on https://github.com/FEniCS/dolfinx

* Some initial spherical .stl files are provided in MESH/2D/. If you want to use other meshes, you can generate them with Gmsh (https://gmsh.info/)
